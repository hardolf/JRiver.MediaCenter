LyricsFinder

The LyricsFinder finds lyrics for the currently playing songs in the "Playing Now" list.

The LyricsFinder can be used as a standalone program and/or as a plug-in for the JRiver Media Center.

Visit the project repository on GitHub: https://github.com/hardolf/JRiver.MediaCenter/tree/master/LyricsFinder

Wiki with user manual: https://github.com/hardolf/JRiver.MediaCenter/wiki/LyricsFinder

Bug reports: https://github.com/hardolf/JRiver.MediaCenter/projects/3

Download area with source code and binaries: https://github.com/hardolf/JRiver.MediaCenter/releases
